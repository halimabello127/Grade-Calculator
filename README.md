# Grade-Calculator
A simple Java program that calculates total score, average, and assigns grades
## 📌 Description
This is a simple Java console application that calculates the total score, average, and assigns a grade based on student performance.

## 🚀 Features
- Accepts input for multiple subjects
- Calculates total and average
- Assigns grades (A, B, C, F)

## 🛠️ Technologies Used
- Java
- Scanner class for input handling

## 📚 What I Learned
- Variables and data types
- Loops (for loop)
- Conditional statements (if/else)
- Basic problem-solving in Java
